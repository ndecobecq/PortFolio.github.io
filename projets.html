Voici mes différents projets

Enigma

    Pour le projet Enigma, nous avons dû créer une simulation de la machine Enigma en IJava langage créé par l'IUT. Ce programme est en affichage terminal et permet d'encoder ou de décoder un message avec le fonctionnement d'Enigma

FreeBit

    FreeBit, c'est le nom d'une entreprise fictive que l'on a dû créer en cours de communication. Pour le projet nous avons dû créer un site internet en HTML et CSS pour cette entreprise. Voici ci-joint la page d'accueil du site.

    FreeBit

Base de données Parcoursup

    Dans ce projet, nous avons récupéré un CSV avec toutes les données parcoursup de l'année 2021, nous avons ensuite dû trier ces données sous la forme d'une base de données en SQL(voir MCD ci-dessus) avec cette base de données nous avons fait plusieurs statistiques centrées sur les départements Normandie et Pays de la Loire(voir le rapport ci-dessous).

    Rapport

Application de tri

    Pour ce projet, le but était de créer une IHM en Java et JavaFX qui permet de rentrer un fichier CSV contenant des données, nous avions par exemple des pokémons, les passagers du Titanic et des Iris. Le programme permet par la suite de regrouper par critères et de les afficher sur un graphique.

APIzzeria

    Pour ce projet, le but était de créer une API Rest pour une pizzeria. Elle permet de gérer les commandes, les recettes, les ingrédients etc...

Agar.io

    Le projet devait être codé en JavaScript et dans ce jeu, on incarne une boule et notre but est de devenir toujours plus gros.

Adopt1Code

    Ce projet effectué durant la startup week est un tinder du code, le but est de donner la possibilité à un utilisateur de partager un extrait de code et à une communauté de pouvoir réagir sur cet extrait afin d'avoir des retours sur son travail.
